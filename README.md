Ford GT MK IV Web

Descripción

Proyecto web dedicado al Ford GT MK IV, un superdeportivo de competición que representa la evolución máxima de la saga Ford GT.

La página presenta información sobre su historia, evolución, especificaciones técnicas y una galería visual de sus diferentes generaciones.

El objetivo del proyecto es aplicar conocimientos de desarrollo web utilizando HTML, CSS y JavaScript, creando una experiencia visual inspirada en el diseño automotriz de alto rendimiento.

Contenido del proyecto

La página cuenta con:

🏁 Página principal con información general del Ford GT.
📖 Historia del Ford GT y su evolución desde el GT40 hasta el Mk IV.
🖼️ Galería de imágenes de diferentes modelos.
⭐ Sistema de modelos favoritos utilizando JavaScript.
🔝 Botón de regreso al inicio con desplazamiento suave.
